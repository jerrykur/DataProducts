About this application

This application display data from the US FEMA Disaster database regarding Disaster Declarations. Using the application the user can select a year and see the disaster declaration for US states as :
  • A histogram showing each state and the number of disaster declarations for the selected year.
  • A map showing the number of disaster declarations for the state in the selected year.
On the histogram additional controls are available to:
  • Specify the sort order for the histogram
  • Enter an alert value that will cause a message to be displayed below the histogram or map if the number of Disaster Declarations meets or exceeds the entered number. If the number of Disaster Declarations exceeds the entered value, the percentage by which it exceeds the alert value will be displayed.

This application was created by Jerry Kurata as part of the Coursera Data Products class.

The application demonstrates how to use Shiny framework to created interactive applications utilizy the Shiny package from RStudio. The package lets users created applications that can be deployed to a Shiny web server. This allows the application to be accessed and used via an application specific URL.
